# 수화 번역기 프로젝트

## 프로젝트 개요
한국 수화 지문자(ㄱ~ㅎ, ㅏ~ㅣ)를 웹캠으로 인식해 텍스트로 변환하는 웹사이트.
청각장애인을 위한 automate-subtitles 사이트의 확장 프로젝트.

## 기술 스택
- HTML + Vanilla JS (프레임워크 사용 금지)
- MediaPipe Hands (손 랜드마크 추출)
- Vercel 배포

## 파일 구조
```
sign-language/
├── CLAUDE.md         # 하네스 규칙 (이 파일)
├── index.html        # 메인 페이지
├── css/
│   └── style.css     # 스타일
└── js/
    ├── app.js        # 메인 로직 (웹캠, MediaPipe 초기화)
    └── classifier.js # 지문자 분류 로직
```

## 코드 규칙
- 모든 주석은 한국어로 작성
- 함수마다 역할 주석 필수
- 전역 변수 최소화, 필요시 명확한 이름 사용
- 에러 발생 시 사용자에게 한국어로 안내

## 기능 요구사항
- 웹캠 실시간 스트림 표시
- MediaPipe로 손 랜드마크 21개 추출 및 캔버스에 시각화
- 좌표 패턴 기반 한국 지문자 분류 (ㄱ~ㅎ, ㅏ~ㅣ)
- 인식된 글자 화면에 실시간 출력
- 인식된 글자 누적해서 단어 조합 가능
- 초기화 버튼으로 텍스트 리셋

## 디자인 규칙
- 배경색: #0D1B2A (네이비)
- 포인트색: #00C2FF (시안)
- 흰색 텍스트
- automate-subtitles 사이트와 통일된 느낌

## 금지사항
- React, Vue 등 프레임워크 사용 금지
- 외부 백엔드 서버 사용 금지 (브라우저에서만 실행)
- jQuery 사용 금지
- 인라인 스타일 최소화 (style.css에 작성)
